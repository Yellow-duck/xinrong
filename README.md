# xinrong
school poject

织梦模板+后台--目录结构：

a   ---> 默认HTML文件存放目录

data  ---> 系统缓存或其它可写入数据存放目录

dede ---> 默认后台管理目录

images ---> 系统默认模板图片存放目录

include ---> 类库文件目录

install ---> 安装程序目录，安装完后可删除

member ---> 会员目录

plus ---> 附助程序目录

special ---> 专题目录

templets ---> 系统默认内核模板目录

uploads ---> 默认上传目录

index.php ---> 网站默认首页

robots.txt ---> 搜索控制文件

tags.php ---> 标签页
